# LocalEventsMapApp

### Application for looking for events in your city

## Tech Stack
- Firebase Realtime Database for remote data storage
- Room Database for local data storage
- Firebath Auth for users authorization
- Caching with Room
- Offline access
- Jetpack Compose
- Yandex MapKit for map with events

## Setup
- You need to get API key of Yandex MapKit and put it in local.properties file
`YANDEX_KEY=your_api_key`
